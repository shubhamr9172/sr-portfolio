# Antigravity Baseline Rules: Portfolio UI & Logic

## 1. Aesthetic Constraints
* You must strictly adhere to a minimalist, high-contrast design. Use Tailwind utility classes like `bg-zinc-950` for backgrounds and `text-zinc-100` for primary text.
* Avoid cluttered UI components. Utilize ample whitespace and padding.
* Interactive elements (buttons, active terminal cursors) should use a subtle gold accent (e.g., `text-amber-400` or `border-amber-500`).

## 2. API Integration Standards
* All connections to NVIDIA endpoints must be handled server-side (in `/api/` routes) to prevent exposing the `NVIDIA_API_KEY` to the client browser.
* Enforce strict JSON formatting for all LLM responses to ensure the frontend can predictably parse and render the data.
* If an API call fails, the UI must gracefully display a stylized error message within the widget, rather than breaking the entire page structure.

## 3. Code Generation Protocol
* Do not leave placeholder comments like `// Add your logic here`. You must implement the full working solution for the API routes.
* Prioritize React functional components and hooks.
